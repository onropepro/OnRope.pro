# Document Management

Store, organize, and track company documents with digital signature capture and compliance monitoring.

## Overview

The Document Management module provides centralized storage for all company documentation with built-in signature tracking. From safety manuals to insurance certificates, every important document is stored, versioned, and accessible. Digital signature capture ensures employees acknowledge policies, creating the audit trail regulators expect.

## Key Features

### Centralized Document Storage
- Upload any document type (PDF, Word, images)
- Organized folder structure
- Version control for document updates
- Search across all documents

### Digital Signature Capture
- Employees sign documents electronically
- Timestamp recorded with each signature
- Signature data stored securely
- Cannot claim "I never saw it" when signature is on file

### Compliance Tracking
- Track which employees have signed which documents
- Identify outstanding acknowledgments
- Generate compliance reports instantly
- Audit-ready documentation

### Certificate of Insurance (COI) Tracking
- Upload insurance certificates
- AI-powered expiry date extraction (using Gemini)
- Automatic warnings for expired or expiring policies
- Red badge/text for policies expiring within 30 days

### Document Categories
- Safety Manuals
- Company Policies
- Training Materials
- Insurance Certificates
- Equipment Documentation
- Regulatory Compliance

## How It Works

### Uploading Documents
1. Navigate to Documents section
2. Click "Upload Document"
3. Select file and enter metadata
4. Choose visibility (all employees, managers only, etc.)
5. Set if signature is required

### Collecting Signatures
1. Assign document to employees for review
2. Employees receive notification
3. Employee opens document and reviews
4. Signs using digital signature
5. Signature and timestamp recorded

### Running Compliance Reports
1. Navigate to Document Compliance
2. Select document(s) to report on
3. View signature status per employee
4. Export as PDF for auditors

## Common Questions

**Q: Can employees sign from mobile devices?**
A: Yes, digital signatures work on any device with a web browser.

**Q: What happens when a document is updated?**
A: New version is created; you can require re-acknowledgment if needed.

**Q: How long are signatures retained?**
A: Indefinitely, as part of your company's compliance record.

**Q: Can I restrict who sees sensitive documents?**
A: Yes, visibility is controlled by permission settings.
