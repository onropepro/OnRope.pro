# Quoting and Sales

Create professional quotes with labor calculations and tax computation for client proposals.

## Overview

The Quoting module enables rope access companies to generate professional, accurate quotes for building maintenance work. Calculate labor costs based on your actual crew rates, add materials, apply taxes, and present polished proposals to clients. Quote data feeds into project management when jobs are won.

## Key Features

### Quote Creation
- Professional quote templates
- Client information autofill
- Building details included
- Custom line items

### Labor Cost Calculation
- Calculate based on crew size
- Factor in hourly rates
- Estimate project duration
- Include travel time

### Material and Equipment
- Add material costs
- Equipment rental fees
- Consumables tracking
- Custom line items

### Tax Computation
- Configurable tax rates
- GST/HST/PST support (Canada)
- Sales tax support (US)
- Tax-exempt options

### Quote Status Tracking
- Draft
- Sent to Client
- Under Review
- Accepted
- Declined
- Expired

## How It Works

### Creating a Quote
1. Navigate to Quotes section
2. Click "New Quote"
3. Select or enter client information
4. Add building/project details
5. Add line items (labor, materials, equipment)
6. Review tax calculations
7. Generate PDF preview
8. Send to client

### Converting to Project
1. When quote is accepted, click "Convert to Project"
2. Quote details transfer to new project
3. Budget baseline established
4. Client and building information carried over

### Quote Templates
Create reusable templates for common job types:
- Window Cleaning quotes
- Building Wash quotes
- Gutter Cleaning quotes
- Inspection quotes

## Financial Controls

### Permission-Based Access
Quote creation and viewing protected by `canAccessFinancials` permission:
- Company owners: Full access
- Operations managers: Configurable
- Technicians: No access (by default)

### Audit Trail
- All quote changes logged
- Version history maintained
- User actions tracked

## Integration Points

- **Project Management**: Accepted quotes convert to projects
- **Client Management (CRM)**: Client data shared
- **Analytics**: Quote-to-win ratio tracking
- **Building Database**: Building information autofill

## Best Practices

### Accurate Labor Estimates
- Base estimates on historical data from similar projects
- Factor in crew skill levels
- Account for access complexities
- Include setup and cleanup time

### Professional Presentation
- Include company branding (White Label feature)
- Clear scope description
- Detailed line items
- Terms and conditions

### Follow-Up
- Track quote status
- Set expiration dates
- Follow up on pending quotes
- Document client feedback

## Common Questions

**Q: Can I duplicate a previous quote?**
A: Yes, clone any existing quote and modify as needed.

**Q: How do I handle quote revisions?**
A: Create a new version - original is preserved for reference.

**Q: Can clients sign quotes electronically?**
A: PDF quotes can be signed and returned; digital signature integration planned.

**Q: What if labor costs change after quoting?**
A: Original quote preserved; track actual vs quoted in project costing.
