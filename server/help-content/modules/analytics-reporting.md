# Analytics and Reporting

Business intelligence dashboards with labor analysis, productivity metrics, and operational insights.

## Overview

The Analytics module transforms your operational data into actionable business intelligence. Track billable vs non-billable hours, monitor employee productivity, analyze project labor costs, and view real-time active worker status. Make data-driven decisions instead of relying on gut feelings.

- Identify improvement opportunities

#### Overtime Analysis
- View overtime by employee

- Identify overtime patterns
- Monitor labor cost impacts

## How It Works

### Viewing Analytics
1. Navigate to Analytics/Reports
2. Select the report type
3. Choose date range
4. Apply filters (employee, project, etc.)
5. View visualizations and data

### Exporting Reports
1. Configure report parameters
2. Click Export
3. Choose format (CSV, PDF)
4. Download file

### Setting Up Dashboards
1. Access Dashboard Configuration
2. Select metrics to display
3. Arrange widgets
4. Save dashboard layout

A: Access controlled by permissions; typically owners and managers.

A: Yes, comparative analysis is available for trend identification.
