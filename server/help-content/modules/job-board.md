# Job Board

Industry-specific job posting and talent marketplace for rope access professionals.

## Overview

Stop posting on Indeed, Craigslist, and Facebook hoping qualified rope techs see your job. OnRopePro's Job Board is a closed ecosystem exclusively for rope access building maintenance. Every employer is verified. Every technician is relevant. No $22 per-application fees. No filtering through 50 resumes to find 2 actual rope techs. No spam.

## The Golden Rule: Closed Garden Ecosystem

```
Value = Verified Employers x Qualified Technicians x Industry Specificity
```

Key principles:
- **Verified Employers**: Only rope access building maintenance companies can post jobs
- **Qualified Technicians**: Every applicant is a rope access professional
- **Industry Specificity**: Focused marketplace creates higher match quality

## Key Features

### For Employers

#### Job Posting
- Create detailed job listings
- Specify certification requirements
- Set location and work type
- Include salary range
- Define experience requirements

#### Talent Browser
Search technicians by:
- Location
- Name
- Certifications (IRATA/SPRAT level)
- Experience level
- Expected pay rate

#### Direct Job Offers
- Send offers directly from Talent Browser
- Link offers to active job postings
- Track offer status (pending, accepted, declined)

#### Application Management
- View all applicants in one place
- Review resumes and certifications
- Contact qualified candidates
- Track hiring pipeline

### For Technicians

#### Profile Visibility Toggle
- Control exactly what employers see
- Opt-in only - full privacy control
- Enable/disable visibility anytime

#### Expected Pay Rate Display
- Set your rate expectations
- Employers see before reaching out
- No wasted interviews on mismatched compensation

#### One-Click Application
- Profile data auto-attaches
- Resume included automatically
- Certifications shared
- Apply in 30 seconds

#### Offer Notifications
- In-portal notifications for offers
- Accept or decline with one tap
- Track all received offers

#### Safety Rating Visible
Individual Safety Rating (ISR) helps employers assess candidate reliability.

## How It Works

### For Employers
1. Navigate to Job Board
2. Click "Post New Job"
3. Complete job details and requirements
4. Publish listing
5. Review incoming applications
6. Or browse Talent Browser to find candidates
7. Send direct offers to promising technicians

### For Technicians
1. Navigate to Job Board in Technician Portal
2. Browse available positions
3. Click "Apply" on interesting jobs
4. Application submitted with your profile
5. Or enable Profile Visibility to receive direct offers
6. Review and respond to offers

## Common Questions

**Q: Is there a fee to post jobs?**
A: Job posting is included in OnRopePro subscription for companies.

**Q: Can technicians apply without making their profile visible?**
A: Yes, applying to a job doesn't require enabling profile visibility.

**Q: How do I know employers are legitimate?**
A: All employers are verified rope access building maintenance companies on OnRopePro.

**Q: Can I see who viewed my profile?**
A: Currently, view tracking is not available.
