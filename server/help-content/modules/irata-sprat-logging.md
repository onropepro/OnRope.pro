# IRATA/SPRAT Task Logging

Track work hours by task type for IRATA and SPRAT certification progression and renewal.

## Overview

The IRATA/SPRAT Task Logging module helps technicians document their work experience for certification requirements. IRATA and SPRAT certifications require documented hours across specific task categories. This module captures that data automatically as technicians work, eliminating paper logbooks and ensuring accurate certification records.

### Automatic Hour Logging
- Hours captured from work sessions
- Task type assigned per session
- Running totals calculated automatically
- Export ready for certification applications

### Experience Level Tracking
- Level 1, 2, and 3 requirements understood
- Hour thresholds monitored
- Certification renewal dates tracked
- Expiry warnings at 60 and 30 days

## How It Works

### For Technicians
1. Clock in to a work session as normal
2. Select the task type from the dropdown
3. Complete your work
4. Clock out - hours automatically logged to that task category
5. View your cumulative hours in the Technician Portal

### For Certification Renewal
1. Navigate to your IRATA Hours summary
2. Review hours by task category
3. Export your work history as PDF
4. Submit with your certification application

## Common Questions

**Q: Do I need to manually enter hours?**
A: No, hours are captured automatically from your work sessions. Just select the task type.

**Q: Can I edit past entries?**
A: Corrections can be made by your operations manager with proper authorization.

**Q: How do I export for IRATA submission?**
A: Use the Export button in your IRATA Hours section to generate a PDF report.

**Q: What if I work multiple task types in one day?**
A: Create separate work sessions for each task type, or split hours during clock-out.
